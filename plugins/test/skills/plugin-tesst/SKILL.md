# Hello World Plugin Test

## Purpose

Hello World Plugin Test provides a simple introduction to the plugin system. When a user asks to test a plugin, this skill demonstrates the basic functionality and usage.

## When to Use This Skill

Activate Hello World Plugin Test when the user:
- Wants to test the basic functionality of a plugin
- Asks for a simple demonstration of plugin capabilities
- Needs a multilingual response in Portuguese, English, and French

## Example Usage
Say hello!
answer: Hello! This is a test plugin. / Olá! Este é um plugin de teste. / Bonjour! Ceci est un plugin de test.
